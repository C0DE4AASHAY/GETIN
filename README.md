- To use this code you should have to goto [Discord Developer Portal](https://discord.com/developers/applications) and Create a Bot 
<br>

- After Creating the Bot Copy the **TOKEN**
<br>

- Now, you have to go to the `main.py` file and find [BOT_TOKEN]
<br>

- Then Replace with the Token you copied from the Portal
<br>

- And make sure you have all the Libraries intalled in your System Path
<br>

- Now Run the main.py or `AutoReload.py` And check Your Server Either Bot is Online or not
<br>
<br>
<br>

**[+] What is** `AutoReload.py`

`AutoReload.py` is a time-saving feature specifically for VS Code users.<br>

- If you make any changes to the bot's code and save it (Ctrl + S),
- AutoReload.py automatically restarts the bot.
- No need to manually restart the script each time you edit!
